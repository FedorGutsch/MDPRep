# Миграция проекта

Проект телеграм-бота был реорганизован и перенесен в отдельную папку `movie_recommendation_bot/`.

## Что изменилось:

1. Все файлы бота перенесены в папку `movie_recommendation_bot/`
2. Данные (CSV) перенесены в подпапку `data/`
3. Пути в коде обновлены для работы с новой структурой

## Старые файлы (можно удалить из корня):

Если вы уверены, что старые файлы больше не нужны, можно удалить из корневой директории:
- `database.py` (старая версия)
- `init_database.py` (старая версия)
- `recommendation_bot.py` (старая версия)

**Новые файлы находятся в `movie_recommendation_bot/`**

## Как использовать:

1. Перейдите в папку проекта:
```bash
cd movie_recommendation_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Инициализируйте БД:
```bash
python init_database.py
```

4. Запустите бота:
```bash
python bot.py
```
